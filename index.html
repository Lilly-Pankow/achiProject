Here are the steps of this project.
I totally forgot about having to use JQuery, making almost everything from scratch.
If anything, I borrowed from people who solved problems similar to mine on StackOverflow
and similar sites. Mainly the code that figures out which element was clicked.
  It took me about three days to make everything, but not putting in long hours each day.
  Also made the images myself. Took at most 30 minutes, mostly to get the colors nice.
  Credit to my younger brother for play testing the final game and finding a bug.
  1. Basic Tic-Tac-Toe
    Uses a table and innerHTML to set the cell to either X or O when clicked on.
    Does not allow for user to overwrite a taken space.
  2. Narrated Tic-Tac-Toe
    3. Same as previous, but has an additional section where winner is announced.
    It also informs users of invalid moves, or if it ends in a stalemate.
  3. Image Tic-Tac-Toe
    Similar to the previous two, but uses an image in place of text.
    It has the appearance of an Achi board but plays as Tic-Tac-Toe.
  4. Achi
    The game that is required.
